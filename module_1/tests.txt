Зарегистрировать нового пользователя
Предусловия:
- Открыть страницу "Войти или зарегистрироваться"
- Выбрать форму "Зарегистрироваться"
Шаги:
- Шаг 1, Заполнить поле "Адрес электронной почты", есть такое поле?, Такое поле существует
- Шаг 2, Заполнить поле "Пароль", есть такое поле?, Такое поле существует
- Шаг 3, Заполнить поле "Повторите пароль", есть такое поле?, Такое поле существует
- Шаг 4, Нажать кнопку "Зарегистрироваться", появляется сообщение "Спасибо за регистрацию!", Сообщение появилось
